# Developer Feed - Technical Assessment

A modern developer news reader built with **React** and **Tailwind CSS**, fetching real-time technical articles from the **Forem API**.

### 🔗 Live Demo
[https://eloquent-lokum-8efeb5.netlify.app/](https://eloquent-lokum-8efeb5.netlify.app/)

### 🚀 Key Features
* **Real-time Data:** Articles are fetched dynamically from the Forem API.
* **State Persistence:** Bookmarks are saved using `localStorage` for a seamless user experience.
* **Responsive UI:** Fully optimized for Mobile, Tablet, and Desktop using Tailwind CSS.
* **Custom Hooks:** Clean and reusable logic for fetching data and managing bookmarks.

### 🛠️ Tech Stack
* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS
* **API Integration:** Forem API (Dev.to)
* **Deployment:** Netlify
